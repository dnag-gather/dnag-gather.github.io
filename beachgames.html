Step 1: Have one person in your group create a new room by selecting one of the following games.
The game's website will open and when they create a new game on this website, a game link will be provided.

Step 2: Have the game organizer copy the game link and paste it into the chat. Make sure to address the message "To Nearby" so that people in the same area can see it.

Step 3: If you want to join the game, do not click the link in the chat! Instead, copy the link from the chat and paste it into the box below. Even if the maximum # of players has been reached, some games (like Catan) allow you to paste a game link and watch other people play.

IMPORTANT! Do not click the link provided in the chat! If you do, the game will open in a new tab and you'll be unable to see your teammates' video while playing the game.
