Step 1: Have one person in your group create a new room by selecting one of the following games. The game's website will open and when they navigate to create a game on this website, a game link will be provided.

Step 2: Have the game organizer copy and paste the game link to the chat. Address the message "To Nearby" so that people in the same space can see it.

Step 3: If you want to participate, copy the link from the chat and paste it in the box below.

IMPORTANT! Do not click the link provided in the chat! This will open the game in a new tab instead of this window, and you'll be unable to see people's videos in this window.
